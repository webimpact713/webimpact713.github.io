HTML

Answer #1: The DOCTYPE declaration, which stands for Document Type Declaration, specifies the version of HTML or XHTML that a web page is using. It is important because it helps browsers to render the page correctly by determining the document type and version. It ensures that the browser uses the appropriate rendering mode, such as standards mode or quirks mode, to display the content as intended by the developer.

Answer #2: <div> is a block-level element used to group elements for styling purposes, while <span> is an inline element used to apply styles to a piece of text within a block-level element.

Answer #3: Both <id> and <class> attributes are used to uniquely identify HTML elements, but they serve different purposes: The id attribute is used to uniquely identify a single HTML element on a page. Each id value must be unique within the document, and it is often used for specific and singular elements that need to be targeted for styling or scripting. The class attribute, on the other hand, is used to define a group of HTML elements that share the same styling or behavior. Multiple elements can have the same class value, allowing for the application of styles or scripts to multiple elements at once.

Answer #4: HTML stands for HyperText Markup Language. It is the standard markup language used to create web pages.

Answer #5: HTML5 introduced several new features such as <canvas>, <video>, <audio>, <article>, <section>, <header>, <footer>, <nav>, <aside>,<figure>, <figcaption>, <datalist>, <progress>, <meter>, <details>, <summary>, etc.

Answer #6: <script> is used to embed JavaScript code in an HTML document, blocking the rendering of the page until the script is fetched and executed. <script async> is used to load the script asynchronously, allowing the page to continue rendering while the script is being fetched.

Answer #7: The alt attribute in the <img> tag specifies an alternate text for an image, which is displayed if the image fails to load or for accessibility purposes for visually impaired users.

Answer #8: The <meta charset="UTF-8"> tag specifies the character encoding for the HTML document. UTF-8 encoding allows the document to support a wide range of characters from different languages.

Answer #9: HTML elements are the building blocks of HTML pages, while HTML tags are used to define and delimit the elements. Tags consist of an opening tag, content, and a closing tag.

Answer #10: You can create a hyperlink in HTML using the <a>(anchor) tag with the href attribute, which specifies the URL of the destination page.

Answer #11: <ol>(ordered list) tag is used to create a numbered list, while <ul> (unordered list) tag is used to create a bulleted list.

Answer #12: <input type="button"> creates a button that can trigger JavaScript functions, while <input type="submit"> creates a button that submits the form data to the server.

Answer #13: The <title>tag specifies the title of the HTML document, which is displayed in the browser's title bar or tab. It also provides a title for the page when bookmarked or shared on social media platforms.

Answer #14: You can create a table in HTML using the <table> tag along with its child elements <tr> (table row), <th> (table header), and <td> (table data).

Answer #15: Padding is the space between the content and the border of an element, while margin is the space between the border of an element and its surrounding elements.

Answer #16: Padding is the space between the content and the border of an element, while margin is the space between the border of an element and its surrounding elements.

Answer #17: Block-level elements start on a new line and take up the full width available, while inline elements do not start on a new line and only take up as much width as necessary.

Answer #18: The GET method sends form data as part of the URL query string, while the POST method sends form data in the HTTP request body. GET requests are limited by the URL length, while POST requests are not.

Answer #19: Semantic HTML refers to using HTML elements that convey meaning beyond their visual presentation. It helps search engines and screen readers understand the structure and context of web content, improving accessibility and search engine optimization (SEO).

Answer #20: The <meta name="viewport"> tag specifies the viewport settings for responsive web design. It controls the width and scaling of the web page on different devices, ensuring proper display and usability across various screen sizes.
